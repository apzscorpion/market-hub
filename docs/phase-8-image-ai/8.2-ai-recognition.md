# 8.2 — AI Recognition (Optional)

## What We're Building

An optional AI feature that helps the wholesaler add products faster. Take a photo of a product → AI suggests the product name, possible category, and text found on packaging.

## Is This Necessary?

**No.** The app works perfectly without this. This is a future enhancement for when:
- The wholesaler has hundreds of products to add
- Manual entry becomes a bottleneck
- You want to impress users with AI capabilities

**Build this only after Phases 0-7 are complete and stable.**

## How It Would Work

```
1. Wholesaler taps "Add Product with Photo"
2. Takes a photo of the product packaging
3. Photo is sent to Google Cloud Vision API
4. Vision API returns:
   - Text found on the packaging (OCR)
   - Labels (e.g., "ice cream", "food", "packaging")
   - Logos (if any brand logos are recognized)
5. App suggests:
   - Product name (from OCR text)
   - Category (from labels)
6. Wholesaler reviews, corrects, fills remaining fields
7. Saves the product
```

## Google Cloud Vision API

### What It Provides

| Feature | What It Returns | Use Case |
|---------|----------------|----------|
| **TEXT_DETECTION** (OCR) | All text in the image | Read product name, flavor, weight from packaging |
| **LABEL_DETECTION** | Object labels ("ice cream", "food") | Suggest category |
| **LOGO_DETECTION** | Brand logos | Identify brand |
| **OBJECT_LOCALIZATION** | Bounding boxes | Find the product in a cluttered photo |

### API Call (via Cloud Function)

Don't call Vision API from the client (API key exposure). Use a Cloud Function:

```typescript
// functions/src/vision.ts
import * as functions from 'firebase-functions'
import vision from '@google-cloud/vision'

const client = new vision.ImageAnnotatorClient()

export const analyzeProductImage = functions.https.onCall(async (data) => {
  const { imageUrl } = data

  const [result] = await client.annotateImage({
    image: { source: { imageUri: imageUrl } },
    features: [
      { type: 'TEXT_DETECTION' },
      { type: 'LABEL_DETECTION', maxResults: 10 },
      { type: 'LOGO_DETECTION', maxResults: 5 },
    ],
  })

  return {
    text: result.fullTextAnnotation?.text ?? '',
    labels: result.labelAnnotations?.map(l => l.description) ?? [],
    logos: result.logoAnnotations?.map(l => l.description) ?? [],
  }
})
```

### Cost

Google Cloud Vision API pricing:
- First 1,000 units/month: FREE
- After that: $1.50 per 1,000 units

For adding products (not high-frequency), this is negligible.

## Name Extraction Logic

OCR returns ALL text on the packaging. The challenge is extracting just the product name.

Strategy:
1. Get full OCR text
2. Split into lines
3. The largest text is usually the product name (Vision API returns text size)
4. Filter out common non-name text: weight, "MRP", batch numbers, barcodes
5. Present top 3 guesses to the wholesaler

This is heuristic and won't be perfect. The wholesaler always has the final say.

## Category Suggestion

Map Vision API labels to the category system:

```typescript
const labelToCategoryMap: Record<string, string> = {
  'ice cream': 'Ice Cream',
  'ice cream cone': 'Ice Cream > Cones',
  'ice cream bar': 'Ice Cream > Bars',
  'frozen food': 'Frozen Snacks',
  'kulfi': 'Frozen Snacks > Kulfi',
}
```

If a label matches, pre-select that category in the form. If multiple match, show options.

## UI Flow

1. On the product create page, add a camera button: "📸 Scan Product"
2. Takes photo or selects from gallery
3. Uploads to Firebase Storage (temporary folder)
4. Calls the Vision Cloud Function
5. Shows results in the product form:
   - Name field pre-filled with OCR suggestion
   - Category pre-selected from labels
   - Image attached
6. Wholesaler reviews and adjusts
7. Saves as normal

Show a small "AI Suggested" badge on auto-filled fields so the wholesaler knows what to double-check.

## Limitations

- OCR accuracy depends on image quality (blurry photos = bad results)
- Malayalam text OCR is less accurate than English
- Products with no text on packaging can't be identified
- Custom/local products won't match any known brand
- The AI is suggestive, never authoritative
