# 5.3 — Bulk Product Operations

## What We're Building

Tools for the wholesaler to manage products in bulk: import from CSV, export to CSV, bulk price changes, and bulk category reassignment.

## CSV Import

### Use Case
The wholesaler has a spreadsheet with all their products. Instead of adding 100 products one by one, they upload a CSV and the system creates all products at once.

### CSV Format

```csv
name,nickname,category,subcategory,unitType,price,aliases
"Vanilla Ice Cream Big Pack","Vanilla Big","Ice Cream","Family Packs","box",250,"vanilla|vanila|വാനില"
"Chocolate Bar Classic","Choco Bar","Ice Cream","Bars","piece",180,"choco|choco bar|ചോകോ"
```

### Import Flow

```
1. Wholesaler clicks "Import Products"
2. File picker opens → select CSV
3. App reads and parses the CSV
4. Shows preview table with all rows
5. Validates each row:
   - Required fields present?
   - Category exists?
   - Price is a number?
   - No duplicate names?
6. Shows validation results:
   - ✅ 45 valid rows
   - ❌ 3 rows with errors (highlighted)
7. User can fix errors or skip invalid rows
8. Click "Import" → creates products in Firestore
9. Progress bar shows import progress
10. Done: "47 products imported, 1 skipped"
```

### Implementation

**File**: `utils/csvParser.ts`

Use the `papaparse` library for CSV parsing (handles edge cases like quoted commas, Unicode).

```bash
npm install papaparse
```

For Firestore writes, use batched writes (max 500 operations per batch):

```typescript
const batch = writeBatch(db)
products.forEach(product => {
  const ref = doc(collection(db, 'products'))
  batch.set(ref, product)
})
await batch.commit()
```

### Alias Format in CSV

Aliases are pipe-separated within the CSV cell: `"vanilla|vanila|വാനില"`

Split on `|` during import.

## CSV Export

Export the entire product catalog (or filtered selection) to CSV.

### Export Flow
1. Wholesaler clicks "Export Products"
2. App reads all products from store
3. Generates CSV string
4. Downloads as file: `products_2026-04-13.csv`

### Implementation

Generate CSV client-side (no server needed):

```typescript
function exportToCSV(products: Product[]): void {
  const headers = ['name', 'nickname', 'category', 'subcategory', 'unitType', 'price', 'aliases', 'active']
  const rows = products.map(p => [
    p.name, p.nickname, getCategoryName(p.categoryId),
    getSubcategoryName(p.categoryId, p.subcategoryId),
    p.unitType, p.price, p.aliases.join('|'), p.active
  ])

  const csv = [headers, ...rows].map(row =>
    row.map(cell => `"${String(cell).replace(/"/g, '""')}"`).join(',')
  ).join('\n')

  const blob = new Blob([csv], { type: 'text/csv;charset=utf-8;' })
  const link = document.createElement('a')
  link.href = URL.createObjectURL(blob)
  link.download = `products_${new Date().toISOString().split('T')[0]}.csv`
  link.click()
}
```

## Bulk Price Update

Change prices for multiple products at once:

1. Select products via checkboxes
2. Click "Update Price"
3. Options:
   - Set specific price (e.g., ₹250 for all selected)
   - Increase by amount (e.g., +₹10)
   - Increase by percentage (e.g., +5%)
   - Decrease by amount or percentage
4. Preview shows old and new prices
5. Confirm → batch update in Firestore

## Bulk Category Change

Move selected products to a different category:

1. Select products via checkboxes
2. Click "Change Category"
3. Select new category and subcategory
4. Confirm → batch update
